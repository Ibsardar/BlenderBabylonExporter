The B.B.Exporter is a system that helps export basic games made in blender G.E. to to Babylon.js.

This is a protoype, and is being considered for further expansion in the future.

Welcome to the Blender-Babylon Exporter!
Json
Utilizes the universality of the .json format. Created when runnning the export script from Blender Game Engine.
Logic Bricks
Includes export support for many common bricks such as keyboard controls, motion actuators, and many more.
Physics
A simplistic physics system, converted from basic B.G.E. properties to CANNON.js and BABYLON.js implementations.
